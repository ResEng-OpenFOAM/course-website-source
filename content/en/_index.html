{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} This course tries to build (or harness) your OpenFOAM skills by getting you involved in programming new solvers and libraries to solve entry-level reservoir engineering problems (flow in porous media). The advantage of this particular field is the lack of official OpenFOAM solvers for its problems, so, you'll have to rely on yourself, and our help, to build capable solvers! {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-server" title="Project-based Experience!" %}} Each small unit ends with project-like assignments. Course parts suggest realistic capstone projects! {{% /blocks/feature %}} {{% blocks/feature icon="fa-question-circle" title="Practical Assignments" url="https://github.com/google/FOAM-School/ResEng-Course-Website" %}} All lessons and assignments encourage usage of cloud environments to accomplish non-trivial tasks But it's up to you to follow along !! {{% /blocks/feature %}} {{% blocks/feature icon="fa-users" title="Collaboration" url="https://twitter.com/docsydocs" %}} Engage in group mini-projects to learn more from the community! {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section color="primary" >}} {{% blocks/feature icon="fa-code" title="No programming experience needed" %}} C++ knowledge is required only when reaching advanced lessons where you actually need to code stuff! {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/gohugoio/hugo" %}} I encourage Pull Requests on **Github** to the [Course Content itself](https://github.com/FOAM-School/ResEng-Course-Website/pulls) and to the [OpenRSR project](https://github.com/FoamScience/OpenRSR/pulls) {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-linux" title="Run on Linux" %}} OpenFOAM is compiled natively for *Nix systems.
